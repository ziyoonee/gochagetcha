가챠 ( 뽑기 ) 정보를 확인 할 수 있는 사이트를 만들거에요
사이트 이름: (가제) gacha Getcha 가챠게챠 - 가챠를 얻었다는 뜻

사이트 기능

- 카테고리 별로 정보를 조회하는 페이지
    - 가챠샵 / 가챠 별 카테고리 조회
    - 가챠샵 클릭시 > 정보 페이지 위치를 지도로 표시 마커는 캡슐 모양 , 가챠샵이 보유하고있는 가챠 정보 노출
- 가챠 클릭시 > 정보페이지, 가챠를 보유하고있는 가챠샵 위치를 지도로표시, 마커는 캡슐모양 , 최저가 랭킹 가챠샵 정보 노출
- 가챠 신상정보를 알 수있는 페이지
- 달,주 별로 반다이의 원가와 사이트 기준 최저가 노출
- 통합 검색기능
- 쿠키 기반 가챠샵 또는 가챠 즐겨찹기
- 즐겨찾기 확인 페이지
- 지도를 통해 한국의 가챠삽을 확인 할 수 있는 페이지 -> 캡슐 모양
- 사람이 직접 가챠삽 정보, 가챠를 추가할 수 있는 페이지
- 관리자 검토후 추가 되는 구조
- 게시판 질문이나 요구사항을 편하게 올릴 수있는구조로 비밀번호를 설정해서 지울수도있게 ( 로그인이 없으므로)
